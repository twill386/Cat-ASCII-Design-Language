Cat ASCII Design Language

Created by: Thomas Williams View On GitHub

Overview

CADL (Cat ASCII Description Language) is a domain-specific programming language designed to describe cats using traits such as mood, ears, tail, and body type, and render them as ASCII art.

This project demonstrates a full programming language pipeline by using a:

  • Lexer
  • Parser
  • Abstract Syntax Tree
  • Interpreter
  • ASCII Renderer

Example CADL Program

cat Miso {
        mood = "sleepy";
        ears = "pointy";
    }
    draw Miso;

This example shows a complete, valid CADL program that can be parsed and executed by the CADL interpreter. It demonstrates the basic structure of the language, including defining a cat, assigning traits, and producing ASCII output through the CADL execution pipeline. Running this program shows how a simple set of instructions is transformed into a visual cat representation. Try running this example program yourself to see the ASCII representation of Miso!
